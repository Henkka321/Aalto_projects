Time Series Data Export

The Time Series export provides a detailed timeline of your tracked activity.

Files Included:
----------

mindfulness_session_YYYY-MM-DD.csv    - Where YYYY-MM-DD is the starting date for the entries in the file.

Each entry has the following values:

    start time                              - Start date and time of the mindfulness session.
    end time                                - End date and time of the mindfulness session.
    heart_rate_summary_average_bpm          - Average heart rate during the mindfulness session.
    heart_rate_summary_start_bpm            - Start heart rate during the mindfulness session.
    heart_rate_summary_end_bpm              - End heart rate during the mindfulness session.
    pause_times_start_time                  - Start time of the pause interval in seconds since the start of the session.
    pause_times_end_time                    - End time of the pause interval in seconds since the start of the session.
    pause_times_start_timestamp             - Start timestamp of the pause interval.
    pause_times_end_timestamp               - End timestamp of the pause interval.
    metadata_update_time                    - Time of the last update of the mindfulness session.
    metadata_session_name                   - Session name of the mindfulness session.
    metadata_type                           - Type of the mindfulness session.
    metadata_description                    - Description of the mindfulness session.
    metadata_media_content_duration_seconds - Duration of the media content played during the session.
    metadata_active_duration_seconds        - Active duration of the mindfulness session.
    metadata_external_id                    - External id of the mindfulness session.
    data source                             - The origin or source of this data.
